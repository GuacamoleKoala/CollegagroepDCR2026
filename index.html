01

Wikidata Bevragen met SPARQL

30 min

We gebruiken de Wikidata Query Service om Linked Open Data op te halen. Je kunt zelf queries schrijven of de AI-prompt uit de repo gebruiken voor snelle resultaten.

SELECT ?item ?itemLabel ?student ?studentLabel WHERE {
  ?item wdt:P106 wd:Q36180 .
  ?student wdt:P1066 ?item .
  SERVICE wikibase:label { bd:serviceParam wikibase:language "nl". }
} LIMIT 100
