Workshop:
Wikidata naar Netwerk

In deze workshop voor de collegawerkgroep digitale collectieregistratie leren we hoe we data uit Wikidata omzetten in betekenisvolle netwerkvisualisaties. We doorlopen de volledige keten: van SPARQL-queries en AI-opschoning tot Gephi-analyse en publicatie.

Bekijk Guido Gezelle Docs Github Repository Bekijk Presentatie

De Workflow

STAP 01

Data Bevragen (SPARQL)

We starten met de bron. Gebruik de onderstaande query om direct data over correspondentie-netwerken uit Wikidata te halen.

SELECT ?item ?itemLabel ?student ?studentLabel WHERE {
  ?item wdt:P106 wd:Q36180 .
  ?student wdt:P1066 ?item .
  SERVICE wikibase:label { bd:serviceParam wikibase:language "nl". }
} LIMIT 100
Open Query Service Tutorial: Wikidata SPARQL
STAP 02

Opschonen & Structureren

Netwerksoftware heeft 'Nodes' en 'Edges' nodig. We gebruiken AI-prompts om de SPARQL-export razendsnel te transformeren naar bruikbare CSV-bestanden.

Nodes.csv Edges.csv
Handleiding: CSV Structuren
STAP 03

Analyse in Gephi

Importeer de CSV's in Gephi voor de eigenlijke visualisatie. Gebruik algoritmes zoals ForceAtlas2 om clusters en hubs in de collectie zichtbaar te maken.

Download .Gephi Project Tutorial: Gephi voor Beginners
STAP 04

Interactieve Publicatie

Het eindresultaat: een interactieve web-visualisatie waarmee de gebruiker zelf door de data kan navigeren.
