Workshop:
Wikidata naar Netwerk

In deze workshop voor de Collegagroep Digitale Collectieregistratie 2026 leren we hoe we data uit Wikidata omzetten in betekenisvolle netwerkvisualisaties. We doorlopen de volledige keten: van SPARQL-queries en data opschoning tot Gephi-analyse en publicatie met Github.

De Workflow

STAP 01

Data Bevragen (SPARQL)

We bevragen Wikidata met behulp van SPARQL Queries om te ontdekken wie gerelateerd is aan Guido Gezelle. Hiervoor gebruiken we de Wikidata Query Service of generatieve AI prompts.

SELECT ?item ?itemLabel ?student ?studentLabel WHERE {
  ?item wdt:P106 wd:Q36180 .
  ?student wdt:P1066 ?item .
  SERVICE wikibase:label { bd:serviceParam wikibase:language "nl". }
} LIMIT 100
Open Query Service Voorbeeldquery Wikidata
STAP 02

Data Opschonen

In Excel, Spreadsheets of OpenRefine herstructureren we de ruwe datatabellen tot bruikbare bestanden: een bestand met Knopen (Nodes) en een bestand met de verbindingen of Randen (Edges).

Nodes.csv Edges.csv
Vind bestanden in Repo
STAP 03

Visualiseren in Gephi

We importeren de CSV-bestanden in Gephi voor de daadwerkelijke visuele analyse. We passen algoritmes (ForceAtlas2) toe en geven knopen een kleur en grootte gebaseerd op eigenschappen zoals geslacht en type verbinding.

Download .Gephi Project Gephi Downloaden
STAP 04

Publiceren met Github

Vanuit Gephi exporteren we de visualisatie via de Sigma Exporter plugin. Vervolgens hosten we deze map (netwerk) online op Github Pages voor een interactieve webervaring.
